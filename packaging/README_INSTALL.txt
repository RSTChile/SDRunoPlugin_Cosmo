Manual installation instructions for Cosmo (SDRuno Plugin)

1) Close SDRuno if it's currently running.

2) Copy Cosmo.dll into the SDRuno Plugins folder:
   C:\Program Files (x86)\SDRplay\SDRuno\Plugins

3) Start SDRuno.

4) In SDRuno, go to the Plugins menu to enable the Cosmo plugin.

Note: You may need administrator privileges to copy files to the Program Files folder.

For questions or support, visit: https://github.com/RSTChile/SDRunoPlugin_Cosmo
